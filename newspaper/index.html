What is Cloud Computing?

Cloud computing is a phrase used to describe events where computer resources are transferred as a service over a computer network connection. A computer network connection is typically the internet. Therefore, cloud computing is a part of computer science that depends on sharing physical and virtual resources, instead of deploying hardware and software. A good analogy is the following, residents use electricity supplied by a power plant instead of running their own generator.
